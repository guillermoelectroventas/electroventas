Detailed Feature List

Here is the detailed list of features for Twilio Whatsapp Integration module.

Enable Whatsapp Messaging from Odooo

  • The module offers Odoo & Whatsapp Messaging integration.
  • You can send messages to customers directly on their whatsapp from Odoo.
  • You need to save the whatsapp number of the customer in his contact.

Set priority for the message gateway as required

  • Set priority of Twilio Whatsapp Gateway as required to send the messages.
  • Set it to '1' for Odoo to automatically choose Twilio Whatsapp to send the messages.

Send Whatsapp messages to single recipient or group

  • The user can send Whatsapp messages from Odoo to individual(s).
  • The recipient can be single contact, multiple individuals or a group.
  • (Groups are created and saved separately using Odoo SMS notification module)

Send Automatic notifications on customer's whatsapp

  • Conditional alerts, if set triggers automatic messages from Odoo.
  • Automatic alerts can be sent on whatsapp message with the module.
  • Conditional alerts include order confirmation messages, shipping alert, invoice alert, etc
  • You need to set appropriate priority to send these message via Whatsapp from Odoo

Check the history and delivery of the sent message

  • The module saves the delivery status of the every sent message.
  • You can see the status as sent, delivered, failed, etc.

Save draft messages to send later with a click

  • You can save draft Whatsapp messages in Odoo to send later.

Save Whatsapp messages templates in Odoo

  • The module facilitates creating message templates for instant use in Odoo.
  • The template can be directly used while sending the whatsapp message to customer(s).

  • Send Whatsapp Messages

Simple configuration settings to integrate Twilio Whatsapp with Odoo

Set priority of the Twilio Whatsapp Gateway For sending the message

The system automatically picks the SMS gateway while sending the automatic messages.

Generating Twilio Credentials

Login to the Twilio console and create a new project to generate API credentials.

Verify your number and choose python as your language.

Then, select ‘Send whatsapp message’ from the list.

Open Sandbox and send Test SMS to the number shown to verify your number.

Once done, go to the Dashboard to get your credentials
